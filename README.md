Spotify Clone 🎵

A frontend clone of Spotify's web player built with pure HTML and CSS — no JavaScript, no frameworks.

Live Demo
https://spotify-clone-o7k7.onrender.com/

Tech Stack

HTML5
CSS3 (Flexbox, position fixed/sticky, media queries)
Font Awesome (CDN)
Google Fonts (Montserrat)

Layout

Sidebar — navigation and library section
Main Content — scrollable playlist cards
Player Bar — fixed at bottom with playback controls and progress bar

Key Concepts

position: fixed for the bottom player bar
position: sticky for the top navigation
Flexbox for all layout sections
Custom styled input type="range" for the progress bar
Font Awesome for sidebar icons, image files for player icons

Deployment

Deployed as a Static Site on Render with auto-deploy from GitHub.
